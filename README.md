# Proyecto_CheerClick_Systems

Objetivo Desarrollar un sistema de información web y móvil para el club deportivo Blood Tigers Cheers, que permita optimizar los procesos relacionados con la organización de horarios (clases, entrenamientos y eventos), el registro y gestión de PQRS, el seguimiento y calificación del rendimiento de los atletas, así como el control de los implementos deportivos. Con el fin de mejorar la organización, el seguimiento de la información y la eficiencia en el desarrollo de estas actividades dentro del club
